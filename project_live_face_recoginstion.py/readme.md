## 🚀 Project Overview

This project leverages the power of OpenCV along with the pre-trained Haar Cascade classifier for real-time face detection.

While Haar Cascades are lightweight and fast, they are not the most efficient or accurate compared to modern deep learning approaches. This project serves as a foundational implementation, focusing on understanding classical computer vision techniques.

### ⚙️ Current Features
- Real-time face detection using webcam input  
- Grayscale processing for improved detection performance  
- Bounding box visualization around detected faces  

### ⚠️ Limitations
- Haar Cascades may produce false positives  
- Performance can degrade under varying lighting conditions  
- Not optimized for high accuracy or large-scale applications  

### 🔧 Future Improvements
I plan to enhance this project by:
- Improving detection efficiency and reducing latency  
- Smoothing frame processing for better real-time performance  
- Exploring advanced models (e.g., DNN-based detectors) for higher accuracy
